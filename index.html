ABOUT ME

I am a highly driven recent chemistry graduate from the University of Bath. My journey into programming and tech really began during my University career whilst completing several modules involving the use of Python in data analysis calculations of a wide variety of chemical problems. In the months following the end of my course, I continued to further my skills by completing courses in Responsive Web Design on freeCodeCamp as well as improve upon the Python knowledge I gained during my studies.

I have gained extensive experience in agile principles whilst training to become a DevOps Consultant with Sparta Global further deepening my knowledge of both development as well as operations.






Projects

Below are some of the projects I worked on during my training with Sparta Global showcasing the different tools and software used to deploy applications and write multi-functional programs.

Containerisation with Docker

In recent years, containerisation has gained massively in popularity due to its functionality of packaging code together and being able to deploy it in any environment. It is a commonly used tool in the DevOps industry as it allows developers to package and deploy code in a very efficient manner.

This project focused on containerising an application within docker containers and utilise a CI CD pipeline in the testing and deployment of said containers on a Virtual Private Cloud setup on AWS.

Airport Booking System with Python and MSSQL

The focus of this project was to design a multi-functional program using Python and MSSQL to create a flight booking system. To achieve this, we implemented OOP into our code, writing several classes for staff members, passengers and the terminal interface.

The project also provided me with the opportunity to develop my experience in agile principles whilst working in a team and following the SCRUM framework.

Orcheastration with Terraform

Terraform is an incredibly powerful infrastructure as code tool which can be used to design and deploy your cloud infrastructure in an automated fashion. This project focused on using Terraform to set up a Virtual Private Cloud on AWS with the necessary infrastructure including Subnets, Network ACLs and Route Tables.

I was then able combine Terraform with tools like Ansible and Packer to create EC2 instances to host an application and db in the public and private subnets of the VPC with their own security groups.
